About CSS

Boxes, Boxes, Boxes!

Every element in HTML is a box. With CSS, we can style both an individual box (element) and groups of boxes (elements). As detailed in the sections below, CSS is used to alter both the presentation and the layout of HTML.

3 Methods of Implementation

There are three ways to style HTML with CSS. They are all listed below, however; the third method, linking an external page to your HTML document, is preffered.

  1. Styling in-line
    In the HTML document, add 'style=" "' inside an element.
    For example:

    <p style="color:red;">This sentence is red.</p>

    There must be a colon after the property and a semi-colon after the value.
  2. Styling in the head of the HTML document
    In the HTML document, add a <style> tag inside of the <head> of the document. Add properies and values inside the tag, following the { property: value; } format.
    For example:
    <head>
    <style>
    .case-example {
    color: red;
    }
    </style>
    </head>
  3. Linking an external style sheet in the head of the HTML document

Color Values

The 'color:' property is used to specify the color of text.

'background-color:' and 'border-color:' are examples of properties that are used to add color to element backgrounds and borders.


The following is a list of all possible color value types, to use with any color properties.

Type Value code format Example
RGB Values rgb( , , );
Select 3 numbers between 0 and 360 to indicate red, green, and blue values		 rgb(231, 229, 226);
This is the gray color displayed in this box, and the entire background of the page.
RGBA Values rgba( , , , );
This is RGB, but with the addition of a 4th number that indicates the alpha, or opacity, of the color. Indicate this number with a decimal between 0 and 1.		 rgb(231, 229, 226, 0.5);
This is the same color as the one above, but with 50% of the opacity.
Hex codes # ;
Use 6 hexadecimal digits to create the color value		 #000000; (black)
#FFFFFF (white)
HSL
(Hue, Saturation, Lightness)		 hsl( , , );
This value requires a number between 0 and 360 to indicate hue, and then two percentages to indicate saturation and lightness.		 hsl(200, 73%, 47%);
This text shows the color of the example.
HSLA
(Hue, Saturation, Lightness, Alpa(Opacity))		 hsla( , , , );
This format is the same as above, but with a fourth number to indicate opacity. This number should be a decimal, between 0 and 1.		 hsla(200, 73%, 47%, 0.5);
This text shows the same color as above, but with 50% opacity.
